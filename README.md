Sample Application built with Backbone.js and Ratchet 2.0
---

Depend on:

* middleman
* [middleman-bower](bbtfr/middleman-bower)
* middleman-deploy
* underscore
* backbone
* zepto
* ratchet

```bash
# Initialize
$ git clone git@github.com:bbtfr/backbone-ratchet.git
$ cd backbone-ratchet
$ bundle install
$ middleman bower install

# Start the server
$ middleman 
```
